## 简介

基于mediasoup开发实时互动应用项目，可用于**在线教育，多人会议，互动游戏等**。

## 快速上手

```sh
# clone the project
git clone https://github.com/GuoJing1995/1995guojing-meidiasoup-app.git

# install client dependency
cd client
npm install or yarn

# run client
npm run dev

# install server dependency
cd server
npm install or yarn

# run server
npm run start

```
